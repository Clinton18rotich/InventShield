## 🚀 Features

### Core Platform
| Feature | Description |
|---------|-------------|
| 📊 **Dashboard** | Manage inventions, drag & drop, stats overview |
| 🔒 **Vault** | Secure invention storage with blockchain anchoring |
| 📝 **Draft Mode** | Work-in-progress inventions with progress tracker |
| 🔍 **Discover** | Browse all public inventions |
| ➕ **Add Invention** | Submit new inventions (anonymous option available) |
| 📋 **Records** | Permanent archive with integrity hash & backup |
| 💰 **Revenue Dashboard** | Track platform earnings & fees |

### Protection & Security
| Feature | Description |
|---------|-------------|
| 🛡️ **Dead Man Switch** | Auto-publish if you go silent |
| 🚨 **Panic Button** | Instant vault lock + contact alerts |
| 🔗 **Blockchain Anchoring** | Immutable proof of invention ($5) |
| 🕵️ **Anonymous Submission** | Hide your identity |
| 📄 **NDA Generator** | Download NDAs for any invention |
| 🔍 **Patent Search** | Prior art search + Google Patents link |

### Funding & Revenue
| Feature | Description |
|---------|-------------|
| 🎯 **Crowdfunding** | Start campaigns, collect contributions (5% fee) |
| ⭐ **Boost Center** | Feature inventions on homepage ($25/week) |
| 💼 **Investor Portal** | Browse inventions, express interest, view ROI |
| 📊 **Funding Calculator** | Estimate investor returns |
| 🆓 **Fee Waiver** | Free access for inventors in need |
| 🎁 **Referral Program** | Earn $5 per referred inventor |

### Community & Collaboration
| Feature | Description |
|---------|-------------|
| 👥 **Contributors** | Add team members with roles & shares |
| 👥 **Team Management** | Add/remove contributors, manage permissions |
| ⭐ **Ratings & Reviews** | Star ratings & written reviews |
| 💬 **Messages** | Chat per invention |
| 📡 **Activity Feed** | Real-time platform event stream |
| 🏆 **Leaderboard** | Top contributors recognition |
| 📤 **Share Hub** | Share to Twitter, WhatsApp, LinkedIn |

### Tools & Utilities
| Feature | Description |
|---------|-------------|
| ⚖️ **Compare** | Side-by-side invention comparison |
| 🔖 **Bookmarks** | Save inventions to watch later |
| 📥 **Export Reports** | Download invention PDF/TXT reports |
| 🎯 **Milestones** | Track invention development goals |
| 📧 **Email Center** | Send & track investor communications |
| 📊 **Analytics** | Charts, category breakdown, revenue |
| 🔔 **Notifications** | Alert history & bell notifications |

### User Experience
| Feature | Description |

### Auth & Profiles

## 💻 Installation

### Prerequisites
- Node.js 16+
- npm 7+

### Local Setup
```bash
# Clone
git clone https://github.com/Clinton18rotich/InventShield.git
cd inventshield

# Install
npm install

# Run
npm run dev

# Build
npm run build
